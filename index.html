CSS Grid Layout Module

Grid layout module is used specifically for rows and columns, making it easier to design web pages without having to use floats and positioning.

Grid Elements

A grid element is consisted of a parent element. A grid layout must have a parent element with the display property set to grid or inline-grid. Direct child-element(s) of the grid container automatically becomes grid items.

The grid-template-columns Property

The grid-template-columns property defines the number of columns in your grid layput, and it can define the width of each column.

The value is a space-separated-list, where each value defines the width of the respective column.

If you want your grid layout to contain 4 columns, specify the width of the 4 columns, or "auto" if all columns should have the same width.

CSS Grid Item

Child Elements (Items)

A grid container contains grid items.

By default, a container has one grid item for each column, in each row, but that can be styled and multiple rows/columns can be spaned.

The grid-column property

The grid-column property defines on which column(s) to place an item. You define where the item will start and where it will end. The grid-column property is a shorthand property for the grid-column-start and the grid-column-end properties.

To place an item, you can refer to line-numbers, or use the keyowrd "span" to define how many items the columns will span.

The grid-row property
